7.4 Getting Help

These links should take you directly to the Mailing lists http://sourceforge.net/mail/?group_id=5997.2 and the Forums http://sourceforge.net/forum/?group_id=599, lists and forums are archived and searchable so if you are lucky someone already had a similar problem. While mails to the lists themselves are delivered promptly their web front end on sourceforge sometimes shows a severe time lag (up to several weeks), if you're seriously using SDCC please consider subscribing to the lists.
