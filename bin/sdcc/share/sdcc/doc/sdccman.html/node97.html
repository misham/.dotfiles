3.17.4 Other libraries

Libraries included in SDCC should have a license at least as liberal as the GNU Lesser General Public License LGPL.

If you have ported some library or want to share experience about some code which f.e. falls into any of these categories Busses (IC, CAN, Ethernet, Profibus, Modbus, USB, SPI, JTAG ...), Media (IDE, Memory cards, eeprom, flash...), En-/Decryption, Remote debugging, Realtime kernel, Keyboard, LCD, RTC, FPGA, PID then the sdcc-user mailing list http://sourceforge.net/mail/?group_id=599 would certainly like to hear about it.

Programmers coding for embedded systems are not especially famous for being enthusiastic, so don't expect a big hurray but as the mailing list is searchable these references are very valuable. Let's help to create a climate where information is shared.
